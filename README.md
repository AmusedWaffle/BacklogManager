# Backlog Manager
A web app to help users track their game libraries and backlog.

## Notes:
1. Backend:
   Flask (Python)
   PostgreSQL (Database)
   SQLAlchemy (ORM)
2. Dependencies:
   flask
   flask_sqlalchemy
   flask_cors
   psycopg2

## Coding Standards:
1. We follow PEP 8 coding standards for Python.
2. Code quality is checked using pylint.
